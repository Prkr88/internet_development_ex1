Q&A


  1. What are the main differences between web1,web2 and web3?
    • Web 1.0 pages were static, which means there was no interaction between the user and the site.
    • In web 2.0 introduced user accounts and profiles , and the users were invited to contibute to the content of the site.
    • Web 2.0 enbaled more features like "ask the web" which contribute greatly to users data share.
    • Web 3.0 took the features of Web2 one step ahead, and its about making the internet more indexed and with unified semantics. Use of AI algorithms became common and users started creating Web API's.

  2. Use case: Searching for a new laptop to buy.
    • web 1.0: I will need to know the url's for the sites i want to buy from.
    • web 2.0: I will use a search engine to search for laptops and get similar results and related searches from other users.
    • web 3.0: I will use google assistent or any other personal agent to search and find the best results for me.
